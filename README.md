# LoL Ban Checker

### Installation

Install [Python](https://www.python.org/downloads/) 3.0+ to run.

Install the dependencies.

```sh
$ pip install requests
```

Run it...

```sh
$ python check.py
-> Username: example_username
-> Password: example_password
```
